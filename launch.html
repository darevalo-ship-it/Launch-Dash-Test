Data Sources

  • Shopify — Orders, line items, products, variants, customers
  • NetSuite — Net sales after returns and discounts applied
  • Snowflake DAASITY_DB · UOS schema — Unified Order Schema combining Shopify + NetSuite
  • Snowflake DAASITY_DB · GSHEETS schema — SKU-level unit launch day forecasts from Google Sheets

Data Freshness

This dashboard uses a static snapshot taken Jun 16, 2026 from Snowflake DAASITY_DB. Data is not live.

To get a refreshed snapshot, Claude must re-query Snowflake and re-generate the HTML. Contact your Data team or re-run the dashboard generation script.

For a live connection, deploy the Node.js server (npm start) which queries Snowflake on each page load.

Metric Definitions

  • Net Sales — SUM(price × quantity − discount_amount) where REFUND_FLAG IS NULL OR FALSE
  • Units Sold — SUM(quantity) on all non-refunded line items
  • AOV — Net Sales ÷ Total Orders (order level)
  • Units / Order — Total Units ÷ Total Orders
  • New Customer — Customer whose order date equals their MIN(order_date) across all orders in UOS.ORDERS
  • Returning Customer — All others (prior purchase history exists)
  • % to Plan (Units) — Cumulative actual units ÷ SUM(plan_units) from GSHEETS.SKU_LAUNCH_DAY_FORECAST through snapshot date

Not Yet Available

  • Conversion Rate — Requires GA4 sessions + transactions
  • Subscriptions — Requires Recharge subscription platform data
  • Traffic Sources — Requires GA4 source/medium dimensions
  • Landing Page Metrics — Requires GA4 page-level events
  • PDP Health — Requires GA4 product detail page events + Shopify
  • Cross-Sell Analysis — Requires basket-level order item pairing

To enable these, the relevant data sources need to be piped into the Snowflake warehouse and surfaced via the server-side API.
